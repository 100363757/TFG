In order to access the functionality of the static websites listed here it may be required to run node js applications.

the pages:
  -pose_reader.html
  -pose_recorder.html

REQUIRE the execution of the node js application backServer.js located in TFG/dataProcessor/backServer.js

the page:
  -pose_predictor.html
 
 does NOT require the execution of any other node js application
 
 
 ALL html files are accessed through a serve by the node js application webServer.js located in TFG/webserver/local-server/webServer.js
